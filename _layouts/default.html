Case Studies

MPs in Hungary are hiding their assets in scanned PDFs preventing the data from being analysed. By using Moonsheep, K-Monitor showed that data liberation is not rocket science. PROS

  • For complex, structured documents that contain labeled and linked information.
  • For a lot of documents.
  • For documents following similar structure in a whole dataset.
  • Best for non-machine readable documents.
  • If you have the data.

CONS

  • There are better tools for tagging images and transcribing texts.
  • If you have little information to transcribe it might be easier to do it by hand.
  • If each document is a different you may want to use Scribe.
  • For copy-pastable PDFs with tables try Tabula.
  • If you need to gather data via FOIA requests try Alavateli.

Analysis

On top of basic reporting that comes with Moonsheep dashboard, we offer coding of custom reports and browsing capabilities. Raw data is rarely goal in itself. We can do a data-analysts and help find the information you are looking for.
